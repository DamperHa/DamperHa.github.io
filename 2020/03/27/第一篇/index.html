Hexo+Git 搭建网络

Posted on 2020-03-27 In 算法

以下部分是简单记录利用Hexo+git建立该网站的步骤！！！

由于Hexo的很多包在国外服务器，因此安装起来会比较慢，建立使用国内的淘宝镜像：

执行该代码后，可以利用cnpm代替npm安装相关包；

Git相关

下载安装Git主要有以下两个原因：

  1. 需要将网站挂载到服务器上；
  2. 在本地使用git bash要比cmd更加便利，也会减少错误的产生；

建立网站与git相关的操作有以下两个：

  1. 利用SSH秘钥建立本地电脑与Github账户的联系；（公钥放置github中）
  2. 配置邮箱和用户名；
  3. 在github创立一个repo，并将repo放置_config中；

Hexo

  1. npm install hexo-cli -g \安装
  2. mkdir my_websit \ 创建一个目录
  3. hexo init
  4. npm install
  5. hexo clean + hexo g
  6. hexo s \ 到这一步可以本地查看
  7. hexo d \ 到这一步可以https://damperha.github.io/查看

为了编辑文档的便利性，安装hexo-admin，
