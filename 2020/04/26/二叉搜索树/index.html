二叉搜索树

Posted on 2020-04-26 Edited on 2020-04-27

文章所有内容来自 labuladong、csnote

这篇文章主要介绍二叉搜索树相关的知识点~

  • 定义

二叉搜索树满足任意节点的值大于等于左子树节点，小于等于柚子树节点；

判断二叉搜索树的合法性

  • 题目

  • 思路

    • 对于二叉搜索树种的结点都有这样性质：小于等于柚子树，大于等于左子树，因此定义两个变量min,max,如果满足性质则：min < root < max
    • 保存临近的最小值和最大值，因为是先判断、后递归的，该结点是否满足二叉搜索树，至于 父亲节点、父亲节点的父亲结点有关；
    • 有四种情况：left—-left、right—-right、left—–right、right—left，这个具体怎么操作，交给遍历规则，则bfs(root->left, min, root), bfs(root->right,root, max);

在BST中删除一个数

  • 题目

  • 思路

    • 这里也是树的重建，因此需要注意树的连接，利用框架，找到对应值；
    • 找到对应值后分三种情况：
      • 对应节点为叶子节点, 返回nullptr
      • 对应节点只包含左子树柚子树中的一个，返回未空节点
      • 含有左右子节点，那就找柚子树的最小值，或左子树的最大值，互换两只，然后删除右子树中的，最小值；
